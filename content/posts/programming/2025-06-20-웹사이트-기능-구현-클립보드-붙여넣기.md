### 프론트엔드(JavaScript)

\-Clipboard API 활용

\-File API로 이미지 처리

\-FormData로 파일 업로드

\-에디터 DOM 조작



### 백엔드(넷리파이 CMS)

\-미디어 업로드 API 연동

\-디렉토리에 파일 저장

\-파일명 중복 방지 로직



### 기대효과

\-컨트롤+V로 게시글 작성폼에서 이미지 붙여넣기 가능

\-업로드 된 이미지는 기존 이미지와 동일한 스타일 적용

\-디렉토리에 실제 파일 저장
